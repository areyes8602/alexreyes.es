Pregunta 7 · Operacions combinades

Dues operacions combinades amb fraccions, parèntesis, potències i arrels.

Puntuació màxima · 2 punts

Recorda l'ordre de les operacions: parèntesis i arrels $\to$ potències $\to$ multiplicacions i divisions (d'esquerra a dreta) $\to$ sumes i restes.

  1. $\dfrac{1}{3} + \dfrac{2}{3}\cdot\left(\left(\dfrac{3}{2}\right)^{2} - \dfrac{5}{8}\right) + \dfrac{4}{6} : \dfrac{8}{5}$ 1 p
  2. $\left[\dfrac{5}{12} - \left(\dfrac{1}{3} + \dfrac{4}{5} : \dfrac{2}{3}\right)\right] + \sqrt{\dfrac{4}{25}}$ 1 p
2n ESO · Sentit numèric Combinades Potències Arrels

Correcció pas a pas

a) $\tfrac{1}{3}+\tfrac{2}{3}\cdot\left(\left(\tfrac{3}{2}\right)^{2}-\tfrac{5}{8}\right)+\tfrac{4}{6}:\tfrac{8}{5}$

Resolem primer el parèntesi interior. Calculem la potència:

$$\left(\dfrac{3}{2}\right)^{2} = \dfrac{9}{4}.$$

Restem dins del parèntesi (mcm$(4,8)=8$):

$$\dfrac{9}{4} - \dfrac{5}{8} = \dfrac{18}{8} - \dfrac{5}{8} = \dfrac{13}{8}.$$

Resolem la multiplicació i la divisió:

$$\dfrac{2}{3}\cdot\dfrac{13}{8} = \dfrac{26}{24} = \dfrac{13}{12},\qquad \dfrac{4}{6}:\dfrac{8}{5} = \dfrac{4}{6}\cdot\dfrac{5}{8} = \dfrac{20}{48} = \dfrac{5}{12}.$$

Sumem les tres fraccions amb denominador $12$:

$$\dfrac{1}{3} + \dfrac{13}{12} + \dfrac{5}{12} = \dfrac{4}{12} + \dfrac{13}{12} + \dfrac{5}{12} = \dfrac{22}{12} = \dfrac{11}{6}.$$
Resultat: $\dfrac{11}{6}$.

b) $\left[\tfrac{5}{12}-\left(\tfrac{1}{3}+\tfrac{4}{5}:\tfrac{2}{3}\right)\right]+\sqrt{\tfrac{4}{25}}$

Resolem el parèntesi interior. Primer la divisió:

$$\dfrac{4}{5}:\dfrac{2}{3} = \dfrac{4}{5}\cdot\dfrac{3}{2} = \dfrac{12}{10} = \dfrac{6}{5}.$$

Sumem dins del parèntesi (mcm$(3,5)=15$):

$$\dfrac{1}{3} + \dfrac{6}{5} = \dfrac{5}{15} + \dfrac{18}{15} = \dfrac{23}{15}.$$

Restem el corxet (mcm$(12,15)=60$):

$$\dfrac{5}{12} - \dfrac{23}{15} = \dfrac{25}{60} - \dfrac{92}{60} = -\dfrac{67}{60}.$$

Calculem l'arrel:

$$\sqrt{\dfrac{4}{25}} = \dfrac{\sqrt{4}}{\sqrt{25}} = \dfrac{2}{5}.$$

Sumem (mcm$(60,5)=60$, $\tfrac{2}{5}=\tfrac{24}{60}$):

$$-\dfrac{67}{60} + \dfrac{24}{60} = -\dfrac{43}{60}.$$

$\gcd(43,60)=1$, així que la fracció ja és irreductible.

Resultat: $-\dfrac{43}{60}$.
